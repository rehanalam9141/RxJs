Filter Operator

Filter items emitted by the source Observable by only emitting those that satisfy a specified predicate.

Example-1

Filter by length

Id Name
{{item.id}} {{item.name}}
Example-2

Filter by Gender

Id Name Gender
{{item.id}} {{item.name}} {{item.gender}}
Example-3

Filter by Nth Item

Id Name Gender
{{item.id}} {{item.name}} {{item.gender}}
